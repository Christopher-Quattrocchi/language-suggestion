Which programming languages should you learn?

Deciding which languages you want to prioritize depends on your goals and interests. This form should help you determine what makes the most sense for you.

HTML

Build the structure of the website, important for front end/visual stuff

CSS

Style HTML, necessary to make a page look not terrible.

JavaScript

Most popular programming language in web development, needed to make sites functional. Used in front and back end. Everyone needs to know it.

Python

Popular, easy to use programming language. Great for automation, data science, used in back end but also somewhat front end.

SQL

Database management, needed for back End

C#

Faster than python, used in bots. Flexible data types. Very versatile. Gaming. One of the most popular in the world. Easy learning curve (relatively)

React

Developed by programmers at Amazon, great for setting up UI

jQuery

JS library for web development. Makes life easier

Java

very transferable language, used across platforms. Mobile and web apps, IoT devices, gaming, big data etc.

Ruby

High level and simple. Web development, DevOps and automation. Data processing. Web scraping.

PHP

Works well with HMTL. Easy to use. Popular among full stack and back end developers.

Rust

High performance. Used to write new Linux kernal. Back end

ASP.net

Server side framework for web development

Go

Server-side (back end) dev, game making, cloud based programming, data science.

Scala

Machine learning, web apps

Swift

Also Apple, more convenient than objective-c. Iphone and Mac app dev

Lisp

Second oldest language in common use. Aviation software.

Node.js

cross-platform, back-end. Lets devs use JS to write command line tools and for server-side scripting.
